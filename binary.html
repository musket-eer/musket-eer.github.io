Binary Trading Automation

This project developed a live trading bot to interact with the Olymp Trade platform, simulating human trading actions.

Project Overview

Binary trading automation project created a live trading bot that interacted with the Olymp Trade platform to simulate a human trader.

Motivation and Objectives

  • Idea conceived due to long hours spent manually trading on the Olymp Trade platform.
  • Goal: Create a trading bot to run from 8 AM to 4 PM, freeing up time for other tasks while the bot trades on my behalf.

Technical Stack

Technologies and Tools Used:

  • Python, PyTesseract, PyAutoGUI, OpenCV, NumPy, Pillow

These libraries enabled GUI simulation to interact with the platform without an API.

Architecture and Design

The architecture was designed to enable efficient, automated trading operations and handle various trading scenarios.

Core Features and Functionality

  • Extract account balance from screen using pixel coordinates.
  • Enter trade amounts by selecting text areas on the screen and typing a floating point number.
  • Simulate mouse movement and clicks to pick trades based on coordinates.
  • Exit trading loop when network slowdowns are detected.
  • Log each trade for performance review and refinement.

These features enabled autonomous trading and real-time monitoring, fulfilling the projectâ€™s main objectives.

Challenges and Solutions

Challenge: Text extraction due to platform UI and fonts.

Solution: Applied color inversions using Pillow libraries to enhance contrast for accurate text extraction.

Challenge: Detecting network slowdowns.

Solution: Measured account balance before and after each trade. No change in balance within the timeframe indicated a network issue.

Testing and Quality Assurance

Tested text extraction on various screenshots to ensure accuracy. Live-tested on a demo trading account, achieving an average daily profit of 2%.

Conclusion

This project is no longer under active maintenance.
