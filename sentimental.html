Suicide Predictor

Project Overview

This suicide predictor project trained a perceptron model on textual data sourced from a Kaggle dataset, containing posts extracted from Reddit labeled as either "suicidal" or "non-suicidal."

Motivation and Objectives

The project was motivated by the need for tools that could help identify and support individuals showing signs of suicidal ideation online.

Goal: Develop an algorithm capable of detecting suicidal tendencies in text data, with a focus on efficient processing and accuracy improvements.

Technical Stack

Technologies and Tools Used:

  • Perceptron Model: Core model for binary classification.
  • Sentiment Analysis: Integrated to enhance model accuracy by providing emotional context.
  • Parallel Processing: Accelerated training time by utilizing parallelism in data processing and model training.
  • Text Preprocessing: Removed special characters and irrelevant data to improve the quality of training data.

Architecture and Design

The architecture incorporated both the perceptron model and sentiment analysis, combining a binary classifier with emotional insights to achieve more accurate predictions.

Challenges and Solutions

  • Text Preprocessing on Noisy Data
    Challenge: Handling diverse and unstructured Reddit text.
    Solution: Employed rigorous text preprocessing to remove noise, making the data more suitable for training.
  • Enhancing Model Accuracy
    Challenge: The perceptron model alone had limitations in accurately capturing complex emotional cues.
    Solution: Combined sentiment analysis with the perceptron to improve accuracy, though future versions may benefit from more sophisticated neural network architectures.

Conclusion
